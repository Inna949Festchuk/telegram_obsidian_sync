## 🏮 Вариант 3: nomic-embed + Qwen 3.5 (Онлайн)

### ✅ Преимущества Qwen 3.5

| Преимущество | Описание |
|--------------|----------|
| 🌐 **Мультиязычность** | Отличная поддержка русского языка |
| 🧠 **Контекст 256K** | Может обработать очень длинные заметки |
| 💰 **Цена** | Дешевле GPT-4, дороже DeepSeek |
| ⚡ **Качество** | Уровень GPT-4 в тестах |
| 🔌 **API** | Доступен через несколько провайдеров |

### 📡 API Провайдеры для Qwen 3.5

| Провайдер | URL | Цена | Лимиты |
|-----------|-----|------|--------|
| **Alibaba Cloud** | dashscope.aliyun.com | $0.50/1M токенов | Стандартные |
| **OpenRouter** | openrouter.ai | $0.50/1M токенов | Есть бесплатный лимит |
| **Together AI** | together.ai | $0.60/1M токенов | $25 бесплатный кредит |

### 🔧 Настройка

#### Шаг 1: Установка Ollama (для эмбеддингов)

```bash
# Скачайте и установите Ollama
curl -fsSL https://ollama.ai/install.sh | sh

# Только модель для эмбеддингов
ollama pull nomic-embed-text

# Запустите Ollama
ollama serve
```

#### Шаг 2: Получение API ключа Qwen

**Вариант A: OpenRouter (рекомендуется)**
```
1. Перейдите на https://openrouter.ai
2. Зарегистрируйтесь
3. Перейдите в Keys → Create Key
4. Сохраните ключ: sk-or-xxxxxxxxxxxxxxxx
```

**Вариант B: Alibaba DashScope**
```
1. Перейдите на https://dashscope.aliyun.com
2. Зарегистрируйтесь (нужен Alibaba аккаунт)
3. API Key Management → Create Key
4. Сохраните ключ
```

#### Шаг 3: Настройка Smart Connections

```
┌─────────────────────────────────────────────────────────┐
│  Smart Connections Settings                             │
├─────────────────────────────────────────────────────────┤
│  API Provider: OpenRouter                               │
│  API Key: sk-or-xxxxxxxxxxxxxxxx                        │
│  Embedding Model: nomic-embed-text (Ollama)             │
│  Embedding API Provider: Ollama                         │
│  Embedding Base URL: http://localhost:11434             │
│  Chat Model: qwen/qwen-3.5-72b-instruct                 │
│  Context Size: 32768                                    │
│  Temperature: 0.7                                       │
└─────────────────────────────────────────────────────────┘
```

#### Шаг 4: Проверка работы

```
1. Откройте Command Palette (Ctrl+P)
2. Введите: "Smart Connections: Chat"
3. Задайте вопрос: "Что я сохранил про Python?"
4. Должен прийти ответ с ссылками на заметки
```

## 💰 Сравнение стоимости API

### Расчёт для вашего экспорта Telegram

**Предполагаемые данные:**
- ~500,000 сообщений из вашего экспорта
- ~100,000 уникальных заметок после группировки по дням
- Средний размер заметки: 500 токенов
- Индексация: 50,000,000 токенов (однократно)
- Чат: ~10,000 токенов/день

| Провайдер | Индексация (однократно) | Чат (месяц) | Итого (первый месяц) |
|-----------|------------------------|-------------|---------------------|
| **Groq** | Бесплатно* | Бесплатно* | **$0** |
| **DeepSeek** | $13.50 | $3.00 | **~$16.50** |
| **Qwen 3.5** | $25.00 | $5.00 | **~$30.00** |
| **OpenAI GPT-4o** | $125.00 | $25.00 | **~$150.00** |

*Groq имеет бесплатный тариф с лимитами (может измениться)

## 🔒 Приватность и безопасность

| Аспект | Локально | Groq | Qwen | DeepSeek |
|--------|----------|------|------|----------|
| **Эмбеддинги** | ✅ На вашем ПК | ✅ На вашем ПК | ✅ На вашем ПК | ✅ На вашем ПК |
| **Чат запросы** | ✅ На вашем ПК | ⚠️ Сервер Groq | ⚠️ Сервер Qwen | ⚠️ Сервер DeepSeek |
| **Данные заметок** | ✅ Не уходят | ⚠️ Уходят на сервер | ⚠️ Уходят на сервер | ⚠️ Уходят на сервер |
| **Юрисдикция** | 🇷🇺 Ваш ПК | 🇺🇸 США | 🇨🇳 Китай | 🇨🇳 Китай |

**Рекомендация:** Если приватность критична — используйте полностью локальную настройку (llama3.2:1b). Если важнее качество ответов — гибридный вариант с DeepSeek/Qwen.
