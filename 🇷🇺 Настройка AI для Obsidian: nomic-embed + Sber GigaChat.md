## 🤖 Что такое GigaChat

**GigaChat** — это большая языковая модель от российской компании **Сбер**, которая:

| Характеристика | Описание |
|----------------|----------|
| **Разработчик** | Сбер (Sber AI) |
| **Поддержка языка** | ✅ Отличная поддержка русского языка |
| **API** | ✅ Доступно через SberCloud |
| **Локализация** | ✅ Российские серверы (важно для приватности) |
| **Контекст** | До 32,000 токенов |
| **Модели** | GigaChat Pro, Max, Lite |

## 🔑 Получение API ключа

### Шаг 1: Регистрация в SberCloud

```
1. Перейдите на https://sbercloud.ru
2. Нажмите "Зарегистрироваться"
3. Заполните данные:
   - Email
   - Телефон
   - Паспортные данные (требуется для верификации)
4. Подтвердите регистрацию через email/SMS
```

### Шаг 2: Активация GigaChat API

```
1. Войдите в личный кабинет SberCloud
2. Перейдите в раздел "AI/ML" → "GigaChat"
3. Нажмите "Подключить сервис"
4. Примите условия использования
5. Дождитесь активации (обычно 5-15 минут)
```

### Шаг 3: Получение учётных данных

```
1. Перейдите в раздел "IAM" → "Приложения"
2. Нажмите "Создать приложение"
3. Заполните:
   - Имя: Obsidian Smart Connections
   - Описание: AI чат с заметками
4. Сохраните:
   - Client ID
   - Client Secret
```

### Шаг 4: Получение токена доступа

**Вариант A: Через веб-интерфейс**
```
1. Перейдите в раздел "GigaChat" → "API Keys"
2. Нажмите "Создать токен"
3. Скопируйте токен (начинается с "Bearer ...")
```

**Вариант B: Через API (автоматически)**
```bash
curl -X POST "https://ngw.devices.sberbank.ru:9443/api/v2/oauth" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -H "RqUID: <uuid>" \
  -d "scope=GIGACHAT_API_PERS" \
  --user "<Client_ID>:<Client_Secret>"
```

## ⚙️ Настройка Smart Connections

### Шаг 1: Установка плагина

```
1. Откройте Obsidian
2. Настройки → Community plugins
3. Browse → Поиск "Smart Connections"
4. Install → Enable
```

### Шаг 2: Настройка эмбеддингов (локально)

```
┌─────────────────────────────────────────────────────────┐
│  Smart Connections Settings                             │
├─────────────────────────────────────────────────────────┤
│  Embedding Provider: Ollama                             │
│  Embedding Model: nomic-embed-text                      │
│  Embedding Base URL: http://localhost:11434             │
└─────────────────────────────────────────────────────────┘
```

**Команды для установки Ollama:**
```bash
# Установка Ollama
curl -fsSL https://ollama.ai/install.sh | sh

# Скачивание модели эмбеддингов
ollama pull nomic-embed-text

# Запуск Ollama
ollama serve
```

### Шаг 3: Настройка чата (GigaChat)

**Вариант A: Через Custom API (рекомендуется)**

```
┌─────────────────────────────────────────────────────────┐
│  Smart Connections Settings                             │
├─────────────────────────────────────────────────────────┤
│  Chat Provider: Custom (OpenAI Compatible)              │
│  API Key: <ваш токен GigaChat>                          │
│  Base URL: https://gigachat.devices.sberbank.ru/api/v1  │
│  Chat Model: GigaChat-Pro                               │
│  Context Size: 32768                                    │
│  Temperature: 0.7                                       │
└─────────────────────────────────────────────────────────┘
```

**Вариант B: Через плагин GigaChat (если доступен)**

```
1. Установите плагин "GigaChat for Obsidian"
2. Настройки → GigaChat
3. Введите Client ID и Client Secret
4. Выберите модель: GigaChat-Pro
```

### Шаг 4: Индексация заметок

```
1. Откройте Command Palette (Ctrl+P)
2. Введите: "Smart Connections: Re-index all notes"
3. Дождитесь завершения (может занять 10-60 минут)
4. Проверьте лог: "Indexed X notes"
```

## 🧪 Тестирование и проверка

### Тест 1: Проверка подключения к GigaChat

```bash
curl -X POST "https://gigachat.devices.sberbank.ru/api/v1/chat/completions" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <ваш токен>" \
  -d '{
    "model": "GigaChat-Pro",
    "messages": [{"role": "user", "content": "Привет!"}],
    "temperature": 0.7
  }'
```

**Ожидаемый ответ:**
```json
{
  "choices": [{
    "message": {
      "content": "Здравствуйте! Чем могу помочь?"
    }
  }]
}
```

### Тест 2: Проверка эмбеддингов (Ollama)

```bash
curl http://localhost:11434/api/embeddings \
  -d '{"model": "nomic-embed-text", "prompt": "Привет, мир!"}'
```

### Тест 3: Чат с заметками в Obsidian

```
1. Откройте любую заметку из Telegram экспорта
2. Откройте Command Palette (Ctrl+P)
3. Введите: "Smart Connections: Chat"
4. Задайте вопрос: "Что я сохранил про Python?"
5. Должен прийти ответ с ссылками на заметки
```

### Тест 4: Поиск похожих заметок

```
1. Откройте заметку
2. В сайдбаре должна появиться панель "Smart Connections"
3. Должны отображаться похожие заметки
4. Кликните на заметку — должна открыться
```

## 🐛 Устранение проблем

### Проблема 1: Ошибка авторизации GigaChat

```
Решение:
1. Проверьте срок действия токена (24 часа)
2. Получите новый токен через API
3. Проверьте Client ID и Client Secret
4. Убедитесь, что сервис активирован
```

### Проблема 2: Медленная работа

```
Решение:
1. Используйте GigaChat-Lite вместо Pro/Max
2. Уменьшите Context Size до 16384
3. Проверьте интернет-соединение
4. Используйте локальные эмбеддинги (Ollama)
```

### Проблема 3: Эмбеддинги не работают

```bash
# Проверьте Ollama
ollama list

# Проверьте модель
ollama run nomic-embed-text "тест"

# Перезапустите Ollama
pkill ollama
ollama serve
```

### Проблема 4: Smart Connections не видит заметки

```
Решение:
1. Проверьте, что vault открыт правильно
2. Нажмите "Re-index all notes"
3. Проверьте, что заметки в формате .md
4. Убедитесь, что frontmatter есть в заметках
```
