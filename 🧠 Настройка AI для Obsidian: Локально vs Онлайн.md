| Параметр | Локально (Ollama) | Онлайн (Groq) |

| **Модель эмбеддингов** | nomic-embed-text | nomic-embed-text (локально) |

## 🏠 Вариант 1: Локальная настройка (8 GB RAM)

### Шаг 1: Установка Ollama

```bash
# Скачайте и установите Ollama
curl -fsSL https://ollama.ai/install.sh | sh

# Проверьте установку
ollama --version

# Запустите Ollama (в фоне)
ollama serve
```

### Шаг 2: Скачивание моделей

```bash
# Модель для эмбеддингов (поиск по заметкам)
ollama pull nomic-embed-text

# Модель для чата (лёгкая, для 8 GB RAM)
ollama pull llama3.2:1b

# Проверьте загруженные модели
ollama list
```

**Ожидаемый вывод:**
```
NAME              ID           SIZE      MODIFIED
nomic-embed-text  abc123...    500 MB    2 minutes ago
llama3.2:1b       def456...    1.3 GB    1 minute ago
```

### Шаг 3: Проверка работы моделей

```bash
# Тест эмбеддингов
ollama run nomic-embed-text "Привет, мир!"

# Тест чата
ollama run llama3.2:1b "Расскажи кратко о себе"
```

### Шаг 4: Установка плагина Smart Connections

1. Откройте **Obsidian**
2. Перейдите в **Настройки → Community plugins**
3. Нажмите **Browse**
4. Найдите **"Smart Connections"**
5. Нажмите **Install → Enable**

### Шаг 5: Настройка Smart Connections

1. Откройте **Настройки → Smart Connections**
2. Заполните настройки:

```
┌─────────────────────────────────────────────────────────┐
│  Smart Connections Settings                             │
├─────────────────────────────────────────────────────────┤
│  API Provider: Ollama                                   │
│  Base URL: http://localhost:11434                       │
│  Embedding Model: nomic-embed-text                      │
│  Chat Model: llama3.2:1b                                │
│  Context Size: 4096                                     │
│  Temperature: 0.7                                       │
└─────────────────────────────────────────────────────────┘
```

3. Нажмите **Re-index all notes**
4. Дождитесь завершения индексации (может занять 5-30 минут)

### Шаг 6: Проверка работы

1. Откройте любую заметку из Telegram экспорта
2. Нажмите **Ctrl+P** → введите **"Smart Connections"**
3. Выберите **"Show similar notes"** или **"Chat with notes"**
4. Задайте вопрос: *"Что я сохранил про Python?"*

## ☁️ Вариант 2: Онлайн настройка (Groq API)

### Шаг 1: Получение API ключа Groq

1. Перейдите на [https://console.groq.com](https://console.groq.com)
2. Зарегистрируйтесь (бесплатно)
3. Перейдите в **API Keys**
4. Нажмите **Create API Key**
5. **Сохраните ключ** (показывается только один раз!)

```
gsk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### Шаг 2: Установка Ollama (только для эмбеддингов)

```bash
# Ollama нужен только для локальных эмбеддингов
curl -fsSL https://ollama.ai/install.sh | sh

# Только модель для эмбеддингов
ollama pull nomic-embed-text

# Запустите Ollama
ollama serve
```

### Шаг 3: Установка плагина Smart Connections

1. Откройте **Obsidian**
2. Перейдите в **Настройки → Community plugins**
3. Нажмите **Browse**
4. Найдите **"Smart Connections"**
5. Нажмите **Install → Enable**

### Шаг 4: Настройка Smart Connections

1. Откройте **Настройки → Smart Connections**
2. Заполните настройки:

```
┌─────────────────────────────────────────────────────────┐
│  Smart Connections Settings                             │
├─────────────────────────────────────────────────────────┤
│  API Provider: Groq                                     │
│  API Key: gsk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxx            │
│  Embedding Model: nomic-embed-text (Ollama)             │
│  Embedding API Provider: Ollama                         │
│  Embedding Base URL: http://localhost:11434             │
│  Chat Model: llama-3.1-8b-instant                       │
│  Context Size: 8192                                     │
│  Temperature: 0.7                                       │
└─────────────────────────────────────────────────────────┘
```

3. Нажмите **Re-index all notes**
4. Дождитесь завершения индексации

### Шаг 5: Проверка работы

1. Откройте любую заметку
2. Нажмите **Ctrl+P** → **"Smart Connections: Chat"**
3. Задайте вопрос: *"Какие каналы про AI я сохранил?"*

## ⚙️ Настройка плагина Smart Connections

### Основные настройки

| Параметр | Значение | Описание |
|----------|----------|----------|
| **API Provider** | Ollama / Groq | Провайдер для чата |
| **Base URL** | http://localhost:11434 | Адрес Ollama |
| **Embedding Model** | nomic-embed-text | Модель для поиска |
| **Chat Model** | llama3.2:1b / llama-3.1-8b-instant | Модель для чата |
| **Context Size** | 4096 / 8192 | Размер контекста |
| **Temperature** | 0.7 | Креативность ответов |

### Дополнительные настройки

```
✅ Enable Smart Connections
✅ Enable Chat
✅ Show similar notes in sidebar
✅ Index all notes in vault
⚠️  Exclude folders: .obsidian, Templates
```

## 🧪 Тестирование и проверка

### Тест 1: Проверка индексации

```
1. Откройте Command Palette (Ctrl+P)
2. Введите: "Smart Connections: Re-index notes"
3. Дождитесь завершения
4. Проверьте лог: должно быть "Indexed X notes"
```

### Тест 2: Поиск похожих заметок

```
1. Откройте заметку из Telegram экспорта
2. В сайдбаре должна появиться панель "Smart Connections"
3. Должны отображаться похожие заметки
```

### Тест 3: Чат с заметками

```
1. Откройте Command Palette (Ctrl+P)
2. Введите: "Smart Connections: Chat"
3. Задайте вопрос: "Что я сохранил про Python?"
4. Должен прийти ответ с ссылками на заметки
```

### Тест 4: Проверка производительности

| Метрика | Локально | Groq |
|---------|----------|------|
| Время ответа | 3-10 сек | 1-3 сек |

## 🐛 Устранение проблем

### Проблема 1: Ollama не запускается

```bash
# Проверьте статус
systemctl status ollama

# Перезапустите
ollama serve

# Проверьте порт
curl http://localhost:11434/api/tags
```

### Проблема 2: Smart Connections не видит заметки

```
1. Проверьте, что vault открыт правильно
2. Нажмите "Re-index all notes"
3. Проверьте, что заметки в формате .md
4. Убедитесь, что frontmatter есть в заметках
```

### Проблема 3: Медленная работа (локально)

```
1. Закройте другие приложения
2. Уменьшите Context Size до 2048
3. Используйте llama3.2:1b вместо 3b/7b
4. Отключите другие плагины на время чата
```

### Проблема 4: Groq API ошибка

```
1. Проверьте API ключ в настройках
2. Проверьте лимиты: https://console.groq.com/usage
3. Попробуйте другую модель: mixtral-8x7b-32768
4. Проверьте интернет-соединение
```

### Проблема 5: Эмбеддинги не работают

```bash
# Проверьте модель эмбеддингов
ollama run nomic-embed-text "тест"

# Проверьте порт Ollama
curl http://localhost:11434/api/embeddings -d '{"prompt": "тест"}'

# Перезапустите Ollama
pkill ollama
ollama serve
```

## 🎯 Рекомендации

### Для 8 GB RAM выбирайте:

| Сценарий | Рекомендация |
|----------|--------------|
| **Приватность важна** | Локально (llama3.2:1b) |
| **Скорость важна** | Онлайн (Groq) |
| **Работа offline** | Локально (Ollama) |
| **Ноутбук на батарее** | Онлайн (Groq) |
| **Много заметок (1000+)** | Онлайн (Groq) |
| **Чат с кодом** | Онлайн (Groq, лучше модели) |

### Гибридный вариант (рекомендуется!)

```
Эмбеддинги: Локально (nomic-embed-text через Ollama)
Чат: Онлайн (Groq API)

Преимущества:
✅ Приватность индексации (данные не уходят)
✅ Скорость чата (мощные модели Groq)
✅ Низкая нагрузка на RAM
✅ Бесплатно (в рамках лимитов)
```
